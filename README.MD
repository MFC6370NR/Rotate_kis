ROTATE

Консольная игра-головоломка.
Поле 4?4 заполняется буквами A–P в случайном порядке.
Задача — упорядочить буквы, вращая квадрат 2?2 по часовой стрелке.

Правила игры

Введите номер верхнего левого угла квадрата 2?2, чтобы повернуть его.

Разрешённые позиции: 1, 2, 3, 5, 6, 7, 9, 10, 11.

Введите -1, чтобы сделать специальный ход (один раз за игру).
Он позволяет поменять местами две соседние по горизонтали буквы.

Введите 0, чтобы выйти из игры.

Как запустить
Linux
make linux
make run-linux

Windows (MinGW/MSYS2)
make windows
rotate.exe

Web (Emscripten)
make web
make run-web

Android (NDK + ADB)
make android
make install-android
make run-android

Очистка
make clean
